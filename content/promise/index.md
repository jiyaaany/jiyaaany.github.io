# Promise 객체
## Promise의 세가지 상태
- pending(대기): 비동기 처리가 아직 완료되지 않은 상태
- fulfilled(이행): 비동기 처리가 완료되어 데이터가 전달된 상태
- rejected(실패): 비동기 처리가 실패되거나 오류가 발생한 상태
